# ⚡ Antigravity Robo-Factory Dashboard
### Premium AI Agent Management System

Este es el panel central de gestión para el **WhatsApp Agent Builder**. Diseñado bajo el estándar **Antigravity Nuclear**, utiliza una estética de "Fábrica de Androides" (White Metal & Robotic Black) para ofrecer una experiencia de usuario de nivel empresarial.

---

## 🎨 Design System: "Robo-Factory"
- **Fondo:** Blanco Metalizado (`#FDFDFD`)
- **Acentos:** Negro Robótico (`#0F172A`)
- **Glow:** Azul Eléctrico (`#3B82F6`)
- **Tipografía:** Outfit (Display) + Inter (Sans)

## 🏗️ Arquitectura
- **Frontend:** Next.js 14 + Tailwind CSS.
- **Iconografía:** Lucide React (Robotic Icons).
- **Flujo:** Cloud-Inject (Cero saturación local).
